一、什么是梅林小宝？它为何如此受欢迎？

“梅林小宝”并非一个独立的软件，而是一款基于**华硕梅林固件（Asuswrt-Merlin）**开发的插件级科学上网管理工具，通常以“脚本+Web前端”的形式存在。它的核心价值，在于将科学上网功能集成到路由器层面，从根本上解决多设备、多终端接入的问题。

为什么选择梅林小宝？

  • 全局代理能力强：部署在路由器层，可为全家所有设备（手机、电视、平板、游戏机）提供稳定代理。

  • 界面友好、操作简单：相比手动配置代理或iptables，梅林小宝提供了简洁直观的可视化操作界面。

  • 插件管理方便：支持在线升级、配置导入导出、定时更新规则等自动化功能。

  • 规则分流灵活：支持GFW列表、国内外域名自动识别分流，有效提高访问效率。

  • 免费开源、社区活跃：有强大的开发者社区支持，教程丰富，问题解决迅速。

二、使用梅林小宝前的准备工作

你需要具备：

  1. 一台支持梅林固件的华硕或刷了梅林固件的路由器
    常见支持机型包括：华硕AC86U、AC68U、AX86U、AX88U、GT-AC2900等。

  2. 可以访问互联网的电脑或手机
    用于下载插件文件、访问管理后台、输入配置信息等。

  3. 一个可用的科学上网节点（代理服务器）
    可为VMess、VLess、Trojan、Shadowsocks等，需提前准备好地址、端口、UUID/密码等信息。

  4. 基础的路由器操作知识
    了解如何进入路由器管理后台、刷固件、安装插件等操作更佳。

三、安装步骤详解：从固件到插件全面部署

Step 1：刷入梅林固件

如果你的路由器尚未安装梅林固件，需要先进行刷机操作：

  1. 访问华硕梅林固件官网或Koolshare论坛；

  2. 下载与你机型对应的固件版本（注意选择稳定版）；

  3. 登录路由器后台（通常是192.168.50.1），进入**“系统管理 > 固件升级”**；

  4. 上传下载的固件文件，等待重启完成；

  5. 成功后界面左下角会显示“Merlin”字样，表明固件已正确安装。

⚠️ 刷机风险需谨慎，操作前务必备份配置，并确保电源稳定。

Step 2：安装Entware环境（插件运行基础）

  1. 登录梅林后台，进入**“软件中心”**；

  2. 安装Entware环境支持（首次使用时系统会自动提示）；

  3. 系统会挂载USB或内部存储空间作为插件依赖目录。

Step 3：安装梅林小宝插件

目前较流行的版本为Koolshare出品的科学上网插件（原梅林小宝），安装方式如下：

  1. 前往Koolshare或GitHub相关页面下载.ipk插件安装包；

  2. 登录路由器后台 → 进入“软件中心” → 选择“手动上传插件”；

  3. 选择刚下载的安装包，点击上传并安装；

  4. 安装完成后，系统将自动在“软件中心”显示科学上网插件（小飞机图标）；

  5. 点击进入插件，即可开始配置。

四、详细配置教程：让你的网络飞起来

1. 添加代理服务器节点

进入梅林小宝界面后，按照以下方式添加代理服务器：

  • 选择**“节点管理”**；

  • 点击“添加节点”；

  • 填入服务器信息（节点名、地址、端口、UUID或密码、协议类型）；

  • 可手动测试连通性并保存。

支持协议包括：

  • Shadowsocks（SS）

  • ShadowsocksR（SSR）

  • V2Ray（VMess/VLess）

  • Trojan（TLS）

2. 设置DNS加速与污染防护

科学上网中DNS设置同样关键：

  • 启用DNS防污染功能；

  • 推荐使用DoH/DoT协议的公共DNS（如Cloudflare 1.1.1.1、Google 8.8.8.8）；

  • 可开启SmartDNS插件，提升域名解析速度并实现自动分流。

3. 配置分流规则

在“访问控制”与“国外网站走代理”栏目中，你可以自由选择：

  • GFW模式：仅代理被墙网站，国内流量直连；

  • 绕过大陆IP模式：所有非中国大陆IP走代理；

  • 全局模式：所有流量均通过代理（适合测试或紧急情况下使用）；

  • 自定义ACL规则：按设备、端口、IP自定义走代理或直连。

4. 启动服务并测试连接

一切设置就绪后：

  1. 返回主界面；

  2. 点击“启动科学上网”；

  3. 页面将显示运行状态、当前流量、DNS状态等实时数据；

  4. 使用手机或电脑访问Google、YouTube等网站测试连接是否成功。

若出现连接失败，请检查网络、节点可用性、DNS设置是否冲突等问题。

五、常见问题与解决方案（FAQ）

Q1：为何配置后无法访问外网？

  • 检查代理服务器参数是否正确；

  • 是否忘记启动插件服务；

  • DNS未配置或被污染导致域名无法解析。

Q2：如何实现按设备分流？

在“访问控制”中添加MAC地址或IP地址，实现精细化管理。

例如：电视全局代理，电脑走GFW模式，打印机则完全直连。

Q3：如何自动更新规则？

插件支持定时自动更新GFW列表与路由表，建议每天更新一次，保障流畅性与有效性。

Q4：梅林小宝插件是否收费？

插件本身为开源免费项目，使用过程中无需额外费用。但代理节点（机场）多数为付费服务。

FAQ

Netch 如何区分全局代理和规则代理？
全局代理会将所有流量通过节点转发，而规则代理仅对指定 IP、端口或应用生效。用户可根据需求选择代理模式，实现流量优化和不同应用的最佳网络体验。
WinXray 节点健康检测如何提升网络体验？
WinXray 会定期检测节点延迟和丢包率，并结合策略组自动切换到最优节点。避免高延迟或失效节点，提高网页浏览、视频播放和游戏连接的稳定性和速度。
V2Ray QUIC 模式适合哪些网络环境？
QUIC 基于 UDP 协议，支持多路复用和零握手，适合高丢包和不稳定网络环境。可提升视频、游戏和即时通讯流量的稳定性和访问速度，提高网络体验。
Matsuri 节点延迟排序功能如何使用？
Matsuri 定期测速节点延迟，并结合策略组自动选择延迟最低的节点。用户无需手动切换节点，即可保持高速访问，提高网页、视频和游戏体验的稳定性。
SSR 混淆方式如何提高抗封锁能力？
SSR 混淆可将流量伪装成正常 HTTPS 或 HTTP 数据，降低被 DPI 检测风险。合理选择协议和混淆参数组合，可以提高节点稳定性和长期可用性，保证科学上网体验顺畅。
QuantumultX 与 Shadowrocket 相比的最大区别是什么？
QuantumultX 在规则系统和自动化脚本上更灵活，支持丰富的 rewrite 与 filter 功能，而 Shadowrocket 则更轻便易用，适合普通用户快速切换节点。二者可根据使用习惯与功能需求选择。
SSR 的协议插件如何应对封锁？
协议插件可隐藏真实通信特征，将流量伪装成正常 HTTP 或 TLS 数据。结合混淆方式使用，可以有效降低被 DPI 检测或封锁的风险，提高节点稳定性和长期可用性。
Sing-box 如何实现域名精确分流？
Sing-box 支持按域名匹配流量，将不同应用流量分配到指定节点。结合策略组使用，可自动选择最优节点，提高访问速度和网络稳定性，实现精确流量管理。
Mellow 的 TUN 模式适合哪些应用？
TUN 模式拦截系统 IP 层流量，可实现全局代理。适合游戏、视频、浏览器和其他全局应用，实现低延迟和稳定连接，保证所有流量通过代理节点，提高访问速度和稳定性。
PassWall2 在路由器中如何更新节点订阅？
在 PassWall2 面板中点击“订阅管理”，输入或更新订阅链接后，点击“手动更新”即可自动下载最新节点数据。也可设置定时任务每日自动更新，保持配置最新。
